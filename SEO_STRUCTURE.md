# SEO_STRUCTURE.md

## Основной принцип

Одна основная услуга — одна отдельная посадочная страница.

Сайт создаётся прежде всего под Яндекс.Директ, но сразу должен быть пригоден для SEO.

## Description

140–170 символов. Включает услугу, регион, одно преимущество и призыв к действию.

## Внутренние ссылки

Страница услуги должна ссылаться на цены, 2–4 связанные услуги, подходящую статью, контакты и форму заявки.

## Контентные кластеры

### Квартиры

Уборка квартир, поддерживающая, генеральная, срочная, сильное загрязнение, после арендаторов, после ремонта.

### Дома

Дома, коттеджи, сезонная уборка, перед заселением.

### Окна

Окна, балконы, лоджии, панорамное остекление, окна после ремонта, витрины.

### Бизнес

Офисы, регулярная уборка, коммерческие помещения, магазины, салоны, склады, мероприятия.

## Техническое SEO

Обязательно: sitemap.xml, robots.txt, canonical, Open Graph, 404, HTTP → HTTPS, единый формат www, единый trailing slash, адаптивность, alt, хлебные крошки, микроразметка, уникальные title, description и H1.

## Микроразметка

Допускаются Organization, LocalBusiness, Service, BreadcrumbList, FAQPage и Article.

Не использовать AggregateRating без реальных отзывов.
